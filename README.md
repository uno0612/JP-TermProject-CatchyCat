### Description
A mini game, Mice Hunt, is a simple aim and click game.  
You can choose difficulty easy, normal, hard.  
Click on tiles to move the cat.  
If the cat lands on a mouse tile, you gain points.  
If the cat lands on a milk tile, the game ends immediately.  
As your score increases, more milk tiles can appear, so the game becomes more dangerous.  
Try to survive as long as possible and beat the high score for that difficulty.  
After losing, you can press the Restart button to play again.  

Click Run then, the rest is intuitive to play. (Also demo gameplay is available in the video)
